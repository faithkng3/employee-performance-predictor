Overview:
SmartHire is a machine learning web application designed to assist HR departments in identifying top-tier talent before they are hired. By analyzing just 6 simple pre-hire indicators, the AI highlights applicants who have a statistically high probability of becoming "Outstanding" performers, reducing manual resume screening time.

Key Features:
Frictionless UI: Only requires 6 inputs (Age, Experience, Education Level, etc.) to generate a prediction.
Bias Mitigation: Intentionally excludes demographic and post-hire data to ensure a purely merit-based screening process.
Explainability: Includes a "Technical Details" expander showing the exact mathematical features passed to the model.
Real-Time Predictions: Generates performance probability scores in milliseconds.

Model:
The core model is a Gradient Boosting Classifier trained on historical employee data.
